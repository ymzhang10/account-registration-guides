## 重要说明

- 本仓库是个人经验总结，不是 Apple、Google、OpenAI 或其他平台的官方说明。
- 账号注册、地区支持、订阅价格、礼品卡渠道、支付方式和风控策略可能随时变化，请以官方页面当前提示为准。
- 不建议买号、接码、代登、代充、共享账号或把账号密码交给第三方。
- 涉及支付、礼品卡、订阅续费和退款时，请自行判断风险。
