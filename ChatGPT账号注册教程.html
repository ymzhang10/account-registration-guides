账号注册教程

账号先稳，订阅才稳。面向中国大陆用户，用长期可控邮箱或手机号路径注册 ChatGPT，避免成品号、共享号、接码号。

核心原则不买号，不共享号，不接码
日常路径邮箱 / Google / Apple / Microsoft 登录
Agent 路径Giffgaff 用户优先手机号注册

账号来源红线

ChatGPT 账号最重要的是长期可控。便宜账号通常把风险转移到你身上。

Account Rules
不买成品号 来路不明、随时找回、容易触发风控。
不用共享号 多人登录会污染行为、付款和安全记录。
不用接码号 后续 OAuth、恢复账号或安全验证时往往无法再验证。
不用代充号 对方可能掌握登录、付款或恢复链路。
不碰永久会员 所谓永久、低价、团队车通常不是稳定个人订阅。

注册前准备清单

先准备邮箱、网络、密码和可恢复的验证方式，再开始注册。

Before Sign Up

网络和地区

使用 OpenAI 支持国家或地区的稳定网络环境，避免频繁切换节点。

邮箱

优先 Gmail；没有 Gmail 时，可用 QQ 邮箱等长期可控邮箱。

登录方式

日常使用可选邮箱、Google、Apple 或 Microsoft 登录；要给 Codex 或第三方 Agent 做 OAuth 时，手机号验证要提前规划。

密码

用密码管理器生成强密码，并保存注册方式，避免下次选错登录入口。

手机号

网页版和 App 日常使用通常不再默认要求手机号，但安全、风控、OAuth 授权时仍可能弹出电话验证。

设备

建议先用电脑浏览器完成注册和安全设置，再登录手机 App。

两条注册路线

普通日常用户走邮箱或社交登录；使用 Giffgaff 号码并且要服务 Agent/OAuth 的用户，优先走手机号注册。

Two Routes

路线 A：邮箱或社交登录

  1. 打开 chatgpt.comchatgpt.com/。
  2. 点击 Sign up。
  3. 选择邮箱注册，或使用 Google / Microsoft / Apple 继续。
  4. 完成邮箱验证。
  5. 进入 Settings，补齐安全设置和恢复方式。
适合：只在网页版和官方 App 日常使用。

路线 B：Giffgaff 手机号注册

  1. 准备可长期接收短信的英国 Giffgaff 号码。
  2. 在注册入口选择手机号方式，而不是先用邮箱注册。
  3. 输入英国号码，完成短信验证码。
  4. 注册完成后，再进入账号设置绑定邮箱。
重点：已用邮箱注册的 ChatGPT 账号，实操中无法顺利补绑 Giffgaff 号码。
手机号注册入口和覆盖国家可能随产品阶段变化。使用 Giffgaff 时，以当前 ChatGPT 注册页面实际可选入口为准。

Codex / 第三方 Agent 的特殊验证

普通登录可用，不代表 OAuth 授权已经满足全部安全验证。

Agent OAuth

核心发现

ChatGPT 注册、网页版登录和 App 登录日常使用通常不再要求手机号码验证；但是账号通过 OAuth 授权给 Codex 或第三方 Agent 使用时，仍可能弹出电话号码验证，并且无法跳过。

如果没有长期可控手机号，这一步会直接卡住。

Giffgaff 处理方式

如果使用英国 Giffgaff 号码，不要先用邮箱注册再补绑手机号。实操结果是：已经完成注册的 ChatGPT 账号无法正常添加该电话绑定。

建议在注册时直接使用手机号注册，完成后再绑定邮箱。

要给 Codex、自动化工具或第三方 Agent 授权的账号，建议一开始就按“手机号注册 -> 再绑定邮箱”的路线设计。

注册后安全设置

账号能登录只是第一步，恢复链路和多因素验证才决定后续是否稳。

Security
保存登录邮箱、登录方式和强密码。
确认可以收到 OpenAI 账号邮件和安全邮件。
开启多因素验证 MFA。
添加 Passkey 或其他更安全的登录方式。
不要把账号交给代充、代登或共享服务。
订阅前确认账号属于自己长期可恢复。

常见问题排查

多数登录问题来自入口选错、邮箱不可控、网络异常或验证链路缺失。

Troubleshooting
问题原因处理
Wrong authentication method曾用 Google / Apple / Microsoft 登录，却改用邮箱密码入口。回到最初的登录方式。
Email already exists邮箱已注册过 ChatGPT 或 OpenAI 账号。直接登录，或用忘记密码流程恢复。
收不到验证邮件邮箱拦截、延迟或地址不可控。查垃圾箱，换 Gmail 或长期可控邮箱。
OAuth 卡电话验证Agent 授权触发额外安全验证。使用长期可控手机号；Giffgaff 用户优先手机号注册路线。
Apple 隐藏邮箱Apple 登录可能生成私有转发邮箱。记录 @privaterelay.appleid.com 地址，避免忘记真实登录凭据。
